## `dir()`

> 組込み関数 [dir()](https://docs.python.jp/3/library/functions.html#dir) は、あるモジュールがどんな名前を定義しているか調べるために使われます。 

> [dir()](https://docs.python.jp/3/library/functions.html#dir) はソートされた文字列のリストを返します

### 引数なし

```python
>>> dir()
['__builtins__', '__doc__', '__loader__', '__name__', '__package__', '__spec__']
```

### 自作モジュール

0/module.py
```python
import module
print(dir(module))
```
0/call.py
```python
def some_method():
    print('some_method()')
```
```sh
$ python3 call.py 
['__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__', 'some_method']
```

何もimportしないときより、以下の名前が増えている。`some_method`は自分で追加した名前である。

* __cached__
* __file__
* __spec__
* some_method
